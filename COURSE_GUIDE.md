# Course Guide

## Overview

This pathway contains 3 courses, each building on React ecosystem skills.

## Course Progression

### 1. React Fundamentals (Beginner â†’ Advanced)
**Focus**: Core React concepts and patterns

**Challenges**:
1. **User Profile Component** (Beginner)
   - Learn: Components, props, state, event handling
   - Skills: Component creation, useState, props

2. **Todo List Application** (Intermediate)
   - Learn: State management, arrays, controlled components
   - Skills: Complex state, array manipulation, conditional rendering

3. **State Management with Context** (Advanced)
   - Learn: Context API, global state, custom hooks
   - Skills: createContext, useContext, Provider pattern, localStorage

**Time Estimate**: 2-3 hours total

### 2. RTK Query (Beginner â†’ Advanced)
**Focus**: Data fetching and state management with Redux Toolkit Query

**Challenges**:
1. **API Setup and Basic Fetching** (Beginner)
   - Learn: RTK Query setup, createApi, fetchBaseQuery
   - Skills: API slice creation, query endpoints, generated hooks

2. **Data Display and Caching** (Intermediate)
   - Learn: Tag-based caching, multiple endpoints, cache invalidation
   - Skills: providesTags, invalidatesTags, tagTypes, refetching

3. **Mutations and Optimistic Updates** (Advanced)
   - Learn: Mutations, optimistic updates, error handling
   - Skills: mutation endpoints, onQueryStarted, optimistic updates

**Time Estimate**: 3-4 hours total

### 3. Next.js App Router (Beginner â†’ Advanced)
**Focus**: Fullstack development with Next.js 13+ App Router

**Challenges**:
1. **Server Components and Basic Routing** (Beginner)
   - Learn: App Router, Server Components, file-based routing
   - Skills: Server Components, Link component, page structure

2. **Data Fetching and API Routes** (Intermediate)
   - Learn: Server Component data fetching, API routes
   - Skills: async components, API routes, error handling

3. **Fullstack Features and Metadata** (Advanced)
   - Learn: Client Components, Server Actions, metadata
   - Skills: 'use client', Server Actions, metadata API, form handling

**Time Estimate**: 3-4 hours total

## Learning Path

### Recommended Order
1. Complete React Fundamentals first (foundation)
2. Then RTK Query (data management)
3. Finally Next.js App Router (fullstack)

### Prerequisites
- Basic JavaScript knowledge
- Understanding of HTML/CSS
- Familiarity with terminal/command line
- Node.js v20+ installed

### After Completion
- You'll have built 3 real applications
- Completed 9 challenges
- Gained comprehensive React ecosystem skills
- Earned pathway badge (Bronze/Silver/Gold)

## Tips for Success

1. **Read Carefully**: Read challenge READMEs thoroughly
2. **Start Simple**: Begin with basic implementation, then enhance
3. **Test Visually**: Always run the app to see your changes
4. **Review Often**: Run `npm run review` frequently
5. **Learn from Feedback**: Check AI feedback for improvement suggestions
6. **Don't Skip Steps**: Complete challenges in order for best learning
